# 🤖 TechBot — AI Technical Mentor

TechBot is a full-stack AI-powered technical assistant built using the MERN stack (MongoDB, Express.js, React, and Node.js). It is designed to help developers and students learn programming concepts, debug code, understand frameworks, and explore technical topics through interactive conversations.

The application uses Anthropic's Claude API to generate intelligent responses while maintaining conversation context across chat sessions. To keep interactions focused and useful, TechBot is restricted to technical and programming-related discussions only.

## ✨ Features

* 💬 Real-time AI conversations powered by Claude Sonnet
* 🧠 Remembers previous messages within a chat session for better context
* 📝 Supports rich Markdown responses including code blocks, tables, lists, and quotes
* 🎨 Clean and modern dark-mode interface built with Tailwind CSS
* 💾 Chat history is stored in MongoDB and can be revisited anytime
* 📂 Sidebar for managing and navigating previous conversations
* ❓ Follow-up learning suggestions after each response
* 🔒 Filters non-technical questions to maintain focus on programming topics
* ⚡ Responsive design that works across desktop and mobile devices
* 📋 One-click copy functionality for messages and code snippets

## 🛠 Tech Stack

### Frontend

* React 18
* Tailwind CSS

* Vite

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Anthropic Claude API

## 🎯 Purpose

The goal of this project is to create a focused technical learning assistant that provides accurate programming guidance, explains concepts in a beginner-friendly way, and helps developers solve real-world coding problems through conversation.
